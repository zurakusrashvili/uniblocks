# uniblocks

[Demo here](http://uniblocks.letsdance.agency)

## Project setup

Install with `npm install uniblocks`.

```
import { uniblocks }  from "uniblocks"
import 'uniblocks/style.css'

<uniblocks v-model="content" mode="json" />
```

Mode can be json or html.
